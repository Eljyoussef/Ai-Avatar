Core Architecture Principle

Event-driven, streaming-first, fully open-source conversational AI avatar platform with document awareness, realtime voice interaction, French multi-accent support, and integrated digital human rendering via Duix.Avatar.

The system is designed around:

ultra-low-latency streaming,
distributed microservices,
modular AI pipelines,
realtime interruption handling,
scalable GPU inference,
self-hosted infrastructure,
avatar-driven conversational experiences.
The Full Flow (9 Stages)
1. User Speech + Avatar UI → WebRTC

Browser captures:

microphone audio,
optional webcam,
avatar interactions.

Streams through:

WebRTC,
Opus codec,
low-latency bidirectional audio/video.

Frontend stack:

Next.js,
React,
WebRTC,
realtime avatar canvas/video player.
2. Gateway → NATS Event Mesh

FastAPI Gateway:

receives WebRTC streams,
authenticates sessions,
manages streaming connections,
converts audio packets into NATS events.

NATS distributes events in parallel across services.

Subjects:

asr.*
rag.*
llm.*
tts.*
avatar.*
session.*
memory.*
metrics.*

This eliminates blocking pipelines and enables realtime concurrency.

3. Session Manager Intercepts First

Dedicated Go/Rust microservice optimized for microsecond latency.

Responsibilities:

session state,
interruption/barge-in handling,
audio playback state,
token buffering,
gender preference,
accent profile,
conversation routing.

Redis-backed hot memory.

Stateless compute layer for easy scaling.

4. Accent Router Classifies

Realtime accent classification using:

XLS-R,
ECAPA-TDNN,
lightweight French accent classifiers.

Supported profiles:

fr-FR,
fr-CA,
fr-AF,
fr-BE,
fr-CH,
multilingual fallback.

Latency target:

<50ms

Routing controls:

ASR profile,
TTS voice embedding,
regional vocabulary adaptation,
pronunciation biasing.
5. ASR Streaming Cluster

Realtime speech recognition powered by:

Faster-Whisper,
CTranslate2,
quantized inference (4-bit/8-bit).

Features:

streaming partial transcripts,
low GPU usage,
realtime decoding,
multilingual support,
French accent fine-tuning.

No sentence completion required before inference begins.

6. RAG Retrieval Mesh (Parallel)

Intent prediction launches retrieval BEFORE speech completion.

Dense Retrieval

Qdrant:

multilingual-e5-large embeddings,
metadata filtering,
semantic memory retrieval.
Sparse Retrieval

OpenSearch:

BM25,
SPLADE,
keyword precision.
Retrieval Fusion

Reciprocal Rank Fusion (RRF).

Reranking

Cross-encoder multilingual reranker.

Context Compression

LLMLingua:

removes filler,
compresses irrelevant context,
reduces token cost.
Optional Knowledge Graph

Neo4j:

document relationships,
semantic references,
chapter linkage,
structured knowledge traversal.
7. vLLM Streaming Inference

LLM inference cluster powered by:

vLLM,
continuous batching,
PagedAttention,
streaming token generation.

Primary models:

Mistral 7B,
Llama 3,
future fine-tuned French conversational models.

Features:

early token streaming,
interruption-aware cancellation,
fallback model routing,
speculative decoding (future optimization).
8. TTS Streaming Cluster

Voice synthesis powered by:

XTTS-v2,
multilingual voice cloning,
streaming phrase generation.

Features:

accent-specific voice embeddings,
realtime chunk synthesis,
voice personalization,
French regional variants.

Fallback chain:

XTTS-v2
   ↓
Kokoro
   ↓
Fish Speech

Premium optional path:

ElevenLabs integration.
